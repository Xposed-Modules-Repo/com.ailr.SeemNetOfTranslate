### **注意事项** 
>1.下载时注意下载对应版本的谷歌翻译

### 功能 
>1.解除谷歌离线翻译在大陆区的限制
>
>2.使用谷歌翻译对屏幕进行翻译

### 使用教程 
>1.Lsp框架打勾谷歌翻译和要翻译的软件(如果不使用屏幕翻译就只需要打勾谷歌翻译)
>
>2.进入谷歌翻译点击编辑框
>
>3.进入Telionslate编辑功能
>
>4.打开需要翻译的软件

### 翻译规则
>如果你不想你开发的应用被翻译
>
>那么你可以创建类 _AILTV_ 继承 TextView
>
>然后在 xml 里面将 TextView 改为 com.xx.xx._AILTV_ 就OK了
